# Prism Console

## Intent
Neon control panel with prism-like gradient pulses.

## Palette
- Primary: Electric violet/blue
- Accent: Success emerald
- Text: Slate/white
- Surface: Deep graphite

## Motion
- Typing pulse
- Active glow
- Working spinner
- Success flash + check

## Accessibility
- Preserve readable contrast for labels and input text
- Maintain visible focus ring/inner glow

## Notes
- Avoid altering the existing gradient sweep timing
