# Crystal Link

## Intent
Refined glassy input with crystalline highlights and clarity.

## Palette
- Primary: Icy blue
- Accent: Emerald success
- Text: Slate/white
- Surface: Frosted glass

## Motion
- Typing pulse
- Active glow
- Working spinner
- Success flash + check

## Accessibility
- Ensure the input text stays readable on glass layers
- Keep focus glow distinct from background

## Notes
- Preserve the shimmer effect and translucent layering
