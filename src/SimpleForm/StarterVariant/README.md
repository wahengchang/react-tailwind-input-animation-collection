# Starter Variant

## Intent
Baseline structure for new variants with neutral styling.

## Palette
- Primary: Soft sky
- Accent: Emerald success
- Text: Slate/white
- Surface: Dark neutral

## Motion
- Typing pulse
- Active glow
- Working spinner
- Success flash + check

## Accessibility
- Maintain readable text contrast
- Keep focus visibility on input states

## Notes
- Use this as the foundation for new Light/Dark pairs
